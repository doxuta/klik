## 3. How It Works

1. **Install**: Use `npm install -g @doxuta/klik` to install CLI globally.
2. **Create**: Use `klik create <name>` to auto-generate a new command.
3. **Run**: Execute with `klik <name>`.
4. **Manage**: View commands via `klik list`.
