## 3. How It Works

1. **Install**  
   ```bash
   npm install -g @doxuta/twap
   ```

2. **Create**  
   ```bash
   twap create <name>
   ```

3. **Run**  
   ```bash
   twap <name>
   ```

4. **Manage**  
   ```bash
   twap list
   ```
