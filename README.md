## 3. How It Works

1. **Install**: Use `npm install -g @doxuta/twap` to install CLI globally.  
2. **Create**: Use `twap create <name>` to auto-generate a new command.  
3. **Run**: Execute with `twap <name>`.  
4. **Manage**: View commands via `twap list`.
